A comprehensive Notes application designed to demonstrate the power of **Supabase** as a backend for **Flutter** apps. This project covers the full lifecycle of data, from secure user sessions to real-time database updates and cloud storage.

## 🌟 Key Features
- **Authentication System:** Secure Sign-up/Login with persistent session management using Supabase Auth.
- **Full CRUD Operations:** Create, Read, Update, and Delete notes with real-time UI synchronization.
- **Media Support:** Integrated Image Picker to upload/delete photos directly from Supabase Storage.
- **Modern UI/UX:** Dark theme implementation with a clean Grid View layout and loading indicators.
- **Network Resilience:** Integrated Internet Connection Checker to handle offline/online states gracefully.

## 🛠️ Tech Stack
- **Frontend:** Flutter (Dart)
- **Backend:** Supabase (Auth, Database, Storage)
- **Database:** PostgreSQL (with Real-time enabled)
- **Architecture:** Clean directory structure for scalability.

## 📂 Project Structure
- `lib/main.dart`: Entry point and routing configuration.
- `lib/screens/`: UI folders (Home, Auth, Note Views).
- `lib/services/`: Supabase integration and authentication logic.
- `lib/component/`: Reusable UI widgets (Buttons, TextFields).
